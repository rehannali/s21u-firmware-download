# S21 Ultra Download Script
Script for download S21 Ultra Firmware using samloader.

Download script to device and give execute rights using `chmod +x download-firmware`

Before using this script, you need to update 2 values in script with your `model` and `CSC`

`DEVICE=SM-G998U`
`REGION=INR`

*You need to update both values maching with your device to work.*
